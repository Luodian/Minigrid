Environment Rule Description

In this environment, the objective is to pick up the single ball in the room as specified in the mission "pick up the ball". The environment is a single room (8x8, 12x12, 16x16, or 20x20 depending on variant), surrounded by walls with no doors or internal structures. The red triangular agent starts in the room facing right. The room contains one ball placed randomly within the room boundaries. Walls block movement and sight. The agent must navigate to the ball and pick it up using the pickup action. Success is achieved by picking up the ball.
