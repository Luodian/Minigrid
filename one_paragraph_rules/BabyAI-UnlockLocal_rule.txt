Environment Rule Description

In this environment, the objective is to open the locked door by first collecting the matching colored key and then using the toggle action on the door. The environment is a single room (default 8x8) with gray outer walls, containing one locked door and one matching key. The red triangular agent starts in the room facing right. The door is initially locked (state=2) and blocks passage. The agent must pick up the key, navigate to the locked door, and use the toggle action to unlock and open it. Success is achieved by opening the door.
