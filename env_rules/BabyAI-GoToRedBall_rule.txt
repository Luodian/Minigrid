Environment Rule Description

In this environment, the objective is to navigate to the red ball as specified in the mission "go to the red ball". The environment is a single 8x8 room surrounded by walls with no doors or internal structures. The red triangular agent starts in the room facing right. The room contains one red ball (the target) and multiple distractor objects in the base variant (various colored objects), while in the NoDists variant, only the red ball is present. Walls block movement and sight. The agent must navigate to the red ball. Success is achieved by reaching the red ball.
